# wxapkg-convertor

一个快速将微信小程序.wxapkg代码包转换成多端框架的工具

## 使用命令行解包

```bash
cd src
node wuWxapkg.js ../wxapkg/小游戏/跳一跳/跳一跳.wxapkg
```
